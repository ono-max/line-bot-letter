Name
====
手紙屋さん(Line-bot)

Overview

This is the line-bot that can creates the easy image of letter.

## Description

This allows user to create the image of letter through the chat.

## Language, Library

Python, Flask

## Requirement

You have to prepare the environment of Python3.

## Usage

If you want to use application as a distribution, you can use pyinstaller.

## Install(terminal)

git clone https://github.com/ono-max/line-bot-letter.git
